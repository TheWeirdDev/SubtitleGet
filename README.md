# SubtitleGet.py

a simple movie subtitle Downloader



# How to use:

`./SubtitleGet.py <Movie name>`

# Example:

`./SubtitleGet.py The Walking Dead` 

This will download the subtitle and moves it to your `~/Downloads` directory
